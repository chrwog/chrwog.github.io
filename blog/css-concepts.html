What is the difference between margin, border, and padding?

11.29.15

Margin, border and padding are three terms that web developers deal with on a daily basis. They are very similar to each other, but represent different parts of the layout of an HTML element. The easiest way to grasp what these terms represent is to look at the CSS box model.

The CSS box model is a visual representation of how CSS wraps around every HTML element:

So what do these terms mean?

  • Content represents the text and images that reside in the box.
  • Padding is a transparent area that clears space around the content. You can keep the padding the same around all sides, or have different amounts of padding on different sides. For example 10px Left padding, and 15px Top padding.
  • The border surrounds the padding and content. It is like a picture frame. It can be different thicknesses, transparent or shown.
  • Finally the margin is like the padding in that it clears space for the HTML element. The difference is that it represents the space around the outside of the border.
